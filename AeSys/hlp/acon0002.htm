Help as a Tutor

You can use the help function as a tutor:

  • Set Help - Always on Top. Use it as a tutor.
  • One hand on mouse, one hand on number keys.
  • Position the cursor with the mouse.
  • In the Pegaesys window , choose Mode-Draw
  • In the Help window, click here --->Draw a Line
  • Follow the steps.
  • Return to Draw Help and try another command.
  • Repeat for all commands, all Modes.

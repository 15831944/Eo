The Windows Clipboard can be used to receive drawing segments.

The trap buffer is used to select segments.

Any program which supports Clipboard can use this function to exchange graphic images.

The Edit functions process the drawing segments as a metafile. If you need the picture in bitmap form, use Print Screen or Alt Print Screen to copy the Pegaesys window to the clipboard.

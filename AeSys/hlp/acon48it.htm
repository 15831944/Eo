G Go-To a Memory Point

Home Points are cursor positions you have set with a numbered stake, or a reference point. You can return to a previously set home point by specifying the stake number. You can also go to other critical points.

Pivot go to the pivot point.

Wrk_origingo to the working origin.

Abs_origingo to the absolute origin.

View_ref go to the view reference point.

Back go to the previous GIN point.

1 2 3 4 ... 9go to a Home Point previously set.

H Home Point Memory

Home Points are cursor positions you can remember by setting a numbered stake, or a reference point, and return to by specifying the stake. Up to 9 numbered stakes can be set.

Pivot to reset the pivot point for edit operations.

Wrk_originto reset the working origin.

View_refto reset the view reference point.

1 2 3 4...9to stake a memory point.

Abs_originto re-register the entire drawing.

You return to a stake by pressing G, the go-to key.

Notes:

Pivot is the same as pressing 1 in Gin Mode: Edit.

Wrk_originis the point from which all position reports annunciate.

Where X Y Z cursor position is reported, it means distance from the working origin.

View_refis the center of the screen.

If you reset the view reference point and then compose a new view, the new view will generate with the view reference point at the center of the screen.

1 2 3 4... 9are stake numbers which you set in a session with the intent of returning to later.

They can be set in overview to pre select pan repaints. They can also be used to recover previous cursor position.

Abs_originis the registration point for the entire drawing. Resetting this parameter in effect moves every element of the drawing.

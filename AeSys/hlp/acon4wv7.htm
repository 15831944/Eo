] Calculator

Calculator provides the function of a programmable calculator, with memory variables, formulas, and recalculation.

Variable display format:

Name = Value or Formula

The calculator display window initially presents only the variables which you have defined.

The reserved variables can be scrolled into the screen.

Reserved Variables:

WkWorking numeric register

WKS Working char. register

DL Diamond Length . or DL=

DA Diamond Angle > or DA=

ELEngaged segment Length Q or EL=

EAEngaged segment Angle Q or EA=

Comments:

If you exit calculator by Esc , the display is erased.

If you exit calculator by Enter, the display remains and is automatically updated.

This is helpful when debugging a User Sequence.

To stop the display, press ] to redisplay calculator, then press Esc to erase.

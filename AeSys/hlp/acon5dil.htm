Overview Window showing 48" X 36" workspace

UpdateRefresh the Keyplan (when displayed)

Check Box

Keyplan Display auxiliary window

The Keyplan is an auxiliary bit map window which provides for rapid composition of a view. The current view is grayed. The next view is previewed by a dynamic outline box. You can drag the box to compose the next view.

Resize the box using the mouse with the Left Key Down or by editing the Zoom Box.

If you click on a button command, the keyplan is ignored.

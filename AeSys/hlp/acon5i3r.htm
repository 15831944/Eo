> (greater) Reset Diamond Angle

This feature resets the reserved variable DimAngZ and changes the diamond Angle report on the message line.

Press > to change the value of DimAngZ.

Set dimension angle in degrees

Enter the desired angle in degrees counterclockwise from the positive X axis.

This directly modifies the angle DimAng z used by the Skew Diamond keys RWZC .

Note that arithmetic expressions are valid: 360/12 6*1.08 etc.

The new angle will be displayed on the message line.

Press R to move the cursor in direction DimAngZ.

Press W,Z, or C for quadrant moves normal to DimAngZ.

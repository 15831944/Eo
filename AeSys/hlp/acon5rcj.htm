. (period) Reset Diamond Length

This feature resets the reserved variable DimLen and changes the diamond Length report on the message line.

Press . to enter a value for DimLen.

Set dimension length in User units

Enter the distance you wish the DIAMOND Keys to move the cursor.

This directly modifies the length the DIAMOND Keys will move the cursor.

Note that arithmetic expressions are valid: 175/21.5 6*1.08 etc.

The new length will be displayed on the message line.

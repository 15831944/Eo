' (apostrophe) enter New Text

Move the cursor to the desired location for the string and press ' .

A dot will appear on the screen marking the location.

Text is composed on the dialog lines using the line editor.

To select character height, and other text attributes, refer to MenuBar-Setup-Note.

To enter fractions in super/subscript: 5^/3/4^

" (quote) Edit Existing Text

Move the cursor to the desired string and press " .

The text string will be appear on the line editor.

The line editor is used for editing existing graphic text.

Attributes are not changed when existing text is edited.

Edit a line of text and post it to the drawing with ENTER.

If paragraphed, the next line is picked up and displayed.

If not paragraphed, a blank line is displayed so that the paragraph can be continued in the attributes of the previously edited line.

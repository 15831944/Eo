N (new attribute) Key

The sequence to change the attributes of a primitive element are:

1). specify the new attributes:

Choose MenuBar-Setup to reset attributes or

Press P (for new pen color) or

Press L (for new line style) ...

A prompt window is displayed

Enter the new value.

2). specify the element to be changed:

Move the cursor to the target element.

Press N

The attributes of the segment under the cursor are changed. You can change attributes of elements on the fly by holding down the N key and passing the cursor thru the elements.

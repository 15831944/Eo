Dimensioned Input

The sequence for entering precise dimension input is

1). specify the length:

Press . (period key)

A prompt window is displayed

Enter the dimension in user units

Graphic tab DL is set.

2). specify the direction.

Press a Diamond Key: ESDX RWZC FV

The cursor moves distance DL in the direction specified by the Diamond Key.

You can change DL and move the cursor on the fly with a GIN operation pending.

for example. if you are in Draw Mode and wish to draw a scaled line from the cursor position 12'6" to the East, press these keys in order:

2.12'6"<Enter>D2

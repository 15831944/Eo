N New Attributes

Move the cursor to the target segment and press N .

The segment will be repainted in the current attributes.

This feature allows you to change the Attribute Options of an existing segment.

The attributes changed are:

TextPen Line Font Incline Spacing Height

ElementsPen and Line

PanelsFill

To change the attributes of a segment or text string:

1.Set the desired Options using the Keys for Pen and Line and MenuBar-Setup.

2.Move the cursor to the target segment and press N .

Note that N changes graphic macros one primitive at a time, not the entire macro at once.

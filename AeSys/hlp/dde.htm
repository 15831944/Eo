Dynamic Data Exchange (DDE) using Visual Basic

DDE is a Windows feature which permits two applications to run simultaneously exchanging data. Visual Basic uses the Properties of a Control to establish a Link with Pegaesys. All three control properties must be included in the Visual Basic code for the application you are writing.

DDE properties supported in Visual Basic:

  • Control.LinkTopic = "Application|Topic"
  • Control.LinkItem = "[a Variable supported by Pegaes]"
  • Control.LinkMode = [an Integer 0-3]

where: Control is a Form, label, picture box or text box and LinkTopic, LinkItem, and LinkMode are Properties of the control.

DDE methods supported in Visual Basic:

  • Control.LinkPoke[???] - to send a value to Pegaes
  • Control.LinkRequest[???]- to get a Pegaesys Variable
  • Control.LinkExecute [???] - to execute a Pegaes command

Shell functions supported in Visual Basic:

Refer to the Visual Basic Manual for supported functions. You can use a shell function to start Pegaesys if it is not already running:

X = Shell("drive:\dir\PEGAES.exe", 4)

Applications in Visual Basic :

Using Visual Basic you can write a applications which, when compiled to a .EXE file, can interact with Pegaesys. Refer to documentation provided by the client software for instructions in setting up an application.

The following sequence is based on an Visual Basic Program as the Source:

  • Open Visual Basic for Windows.
  • Create a <FileName>.MAK program file
  • Create a <FormName>.FRM with controls which can include the DDE commands as Code.
  • Write the code for the DDE commands.
  • Open Pegaesys in a window.
  • Run the program using the Pegaesys - File - Run command from the Bar Menu.

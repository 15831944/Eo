DDE LinkExecute Example 1 - TextBox:

Fragment of code for command for a form with one TextBox and a CommandButton: The TextBox contains a supported command such as [Pen(3)]:

  • Sub Command1_Click ()
  • Dim Cmd As String
  • Cmd = Text1.Text
  • Cmd = Trim(Cmd)
  • Control.LinkTopic = "Pegaes|Command"
  • Control.LinkMode = 2
  • Control.LinkExecute Cmd
  • End Sub

The command from the application Window is executed in the Pegaesys window.

for example, the following program fragment prints a Note to the drawing:

  • Text1.Text = "Hello there"
  • mString = Text1.Text
  • Text1.LinkExecute [note(mString)]

for example, the following program fragment changes the current Pen to Pen 4:

  • Control.LinkExecute [pen(4)]

for example, the following program fragment gets the EngLen report from Pegaes:

  • Sub Command1_Click ()
  • Command1.LinkMode = 0 'off temporarily
  • Command1.LinkTopic = "Pegaes|General" 'initiate Link
  • Command1.LinkMode = 2 'manual
  • Command1.LinkExecute "Q" 'get a EngLen report
  • Text1.LinkTopic = "Pegaes|General"
  • Text1.LinkItem = "EngLen"
  • Text1.LinkMode = 1

Note that the code for the above program does not have to be written; you can set the properties of Command1 and Text1 when you create the form. In this case, only the LinkExecute command would have to be coded for Sub Command1_Click ()

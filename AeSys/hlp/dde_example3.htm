DDE LinkExecute Example 3 - Menu Commands

Pegaesys recognizes menu commands in the format:

Control.LinkExecute [Menu_command(String_data)]

for example, the following program fragment opens layer B:\DIR1\SAMPLE.JB1

  • mLayer = 'SAMPLE'
  • mLayer = Trim(mLayer)
  • mLayer = "B:\\DIR1\\" + mLayer + ".JB1"
  • control.LinkExecute [fileOpen(mLayer)]

where

  • The syntax [???(????)] must be strictly followed.
  • Menu commands supported are: TracingOpen, TracingMap, TracingView, TracingBlank, FileGet
  • String data can be a variable or contents of a TextBox.
  • Since blanks are not allowed, strings must be trimmed
  • Non-ASCII characters and \ must be preceded by the \ character.
  • On Menu commands, control is automatically transferred back to the application window.

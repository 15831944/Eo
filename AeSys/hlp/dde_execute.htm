DDEExecute Function

The DDEExecute function sends a command to PegAEsys, using dynamic data exchange (DDE). The commands recognized by PegAEsys include all defined keyboard keys. This includes most alpha keys and the Drawing Modes numeric keys. In addition a small set of named commands are recognized.

=DDEExecute(channel, command)

where

  • channelis a previously initiated channel
  • commandis a keyboard key command or Named command recognized by Pegaesys

Examples of keyboard key commands

to snap the cursor to a primitive control point:

=DDEExecute(channel, "A")

to move the cursor south:

=DDEExecute(channel, "X")

to change focus to PegAEsys and open the Pen Select dialog:

=DDEExecute(channel,"P")

Examples of Named commands

to changes the current Pen to Pen 4:

=DDEExecute(channel, '[pen("4")]')

to add a Note primitive to the active layer on the active drawing:

  • store 'Hello there' to mString
  • =DDEExecute(channel,'[note("&mString")]')

to open a layer named B:\DIR1\SAMPLE.JB1 in a new window:

  • store 'SAMPLE' to MLAYER
  • store trim(MLAYER) to MLAYER
  • store 'B:\\DIR1\\'+MLAYER+'.JB1' to MLAYER
  • =DDEExecute(channel,'[fileOPEN("&MLAYER")]')

Since blanks are not allowed, strings must be trimmed. Non-ASCII characters and \ must be preceded by the \ character.

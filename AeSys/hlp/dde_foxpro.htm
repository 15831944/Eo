Dynamic Data Exchange (DDE) using Visual FoxPro

DDE is a Windows feature which permits two applications to run simultaneously exchanging data.

Pegaesys supports DDE in server mode. Any application which supports DDE in client mode can interact with Pegaesys. This includes Excel and Visual Foxpro. Refer to documentation provided by the client software for instructions in setting up an application.

The following sequence is based on Visual FoxPro as the client.

  • Open Pegaesys in one window.
  • Open Visual FoxPro in another window.
  • Create a .PRG program file which can include the DDE commands.
  • Run the program file using the FoxPro DO command.

DDE commands supported in Visual FoxPro programs:

  • DDEInitiate - to open a channel
  • DDEExecute- to execute a command in Pegaesys
  • DDEPoke - to send FoxPro variable to Pegaesys
  • DDEAdvise - to update FoxPro on Pegaesys variable change
  • DDERequest- to send a Pegaesys variable to FoxPro
  • DDETerminate- to close a channel

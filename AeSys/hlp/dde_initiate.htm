DDEInitiate Function

In the .PRG file you must start a conversation with the server before data is exchanged. To initiate a conversation with Pegaesys, set up a channel to the server in the format:

Channel = DDEInitiate(ServiceName, TopicName)

where

  • Channelreturns a unique value for DDE calls
  • DDEInitiateis the command
  • ServiceNamein our case is Pegaes
  • TopicNameis: System, General, or Commands.

for example:

iCh1 = DDEInitiate("Pegaes", "General")

iCh2 = DDEInitiate("Pegaes", "Commands")

If Pegaesys program is not already running, it must be started. Any convenient Channel name can be used. Multiple channels can be open simultaneously.

You have now started a conversation between FoxPro and Pegaesys which allows you to pass data and commands between them.

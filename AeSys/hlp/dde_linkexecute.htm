DDE LinkExecute

You can execute Pegaesys commands from your application in the format:

  • Control.LinkTopic = "Pegaes|Command"
  • Control.LinkMode = 2
  • Control.LinkExecute [PegCommand (parameter)]

where

  • LinkExecute is a Visual Basic KeyWord
  • ..PegCommand is a Pegaesys Keyword.
  • Parameter is a string value known to Pegaesys

DDE LinkPoke

You can poke a value to Pegaesys from your application in the format:

  • Control.LinkTopic = "Pegaes|General"
  • Control.LinkItem = Variable
  • Control.LinkMode = 2
  • Control.LinkPoke [(parameter)]

where

  • LinkTopic is a Property of the control
  • LinkItem is a variable which Pegaesys recognizes.
  • LinkMode is a Property of the control,
  • LinkPoke is a Visual Basic KeyWord
  • parameter is a string value for the Variable

Example using a TextBox:

  • Text1.LinkTopic = "Pegaes|General"
  • Text1.LinkItem = "DimLen"
  • Text1.LinkMode = 2
  • Text1.LinkPoke

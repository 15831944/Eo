DDE LinkRequest

You can request a value from Pegaesys for your application in the format:

  • Control.LinkTopic = "Pegaes|General"
  • Control.LinkMode = 1
  • Control.LinkItem = Variable
  • Control.LinkRequest

where

  • LinkTopic is a Property of the control
  • LinkMode is a Property of the control
  • LinkRequest is a Visual Basic KeyWord
  • Variable is a variable which Pegaesys recognizes.

Example using a TextBox:

  • Text1.LinkTopic = "Pegaes|General"
  • Text1.LinkItem = "Scale"
  • Text1.LinkMode = 1
  • Text1.LinkRequest

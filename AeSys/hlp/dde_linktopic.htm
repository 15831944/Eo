DDE LinkTopic

In the Visual Basic program you must start a conversation with Pegaesys before data is exchanged. To initiate a conversation with Pegaesys, set up a Link to Pegaesys in the format:

Control.LinkTopic = "Application|Topic"

where

  • LinkTopic is a Property of the control,
  • Application in our case is Pegaes
  • Topic can be: System, General, or Commands.

for example:

Text1.LinkTopic = "Pegaes|General"

  • The bar | separator is required syntax.
  • The quote marks are required.
  • Pegaes is the server application.
  • General, System, or Commands are server Topics.

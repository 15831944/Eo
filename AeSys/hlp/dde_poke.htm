DDEPoke Function

You can send data to Pegaesys from FoxPro in the format:

=DDEPoke(Channel,"PokeVariableName", "PokeVariableValue")

where:

* "PokeVariableName" is a Pegaes defined topic name.

* "PokeVariableValue" can be composed using the &Macro Substitution function of FoxPro.

for example, the following program fragment changes the Scale in the Pegaesys window:

store "SCALE" to mName

store "48" to mValue

=DDEPoke(iCh1,"&mName","&mValue")

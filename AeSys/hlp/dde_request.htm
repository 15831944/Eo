DDERequest Function

You can request data to Foxpro from Pegaesys in the format:

=DDERequest(Channel,"ItemName")

where

* The Channel must have been previously initiated.

* "ItemName" is a Pegaes defined topic name.

* The function in the Pegaesys window is executed in the FoxProw Window.

for example, the following program fragment reports the Engaged Length to the FoxPro window:

p_EL = DDERequest(iCh1,"EngLen")

? "P_EL is now " + p_EL

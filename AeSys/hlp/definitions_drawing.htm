Drawing

A generic reference to files containing graphic segments.

There are 3 types of drawing files:

Tracing.jb1 is a graphic file with one layer, registered at the absolute origin.

Sheet.jb0 is a graphic file with more than one layer, registered at the absolute origin.

A Sheet can include multiple Tracings using pointers.

Reference.lb1 drawing is a graphic file with one layer,unregistered; a detail drawing.

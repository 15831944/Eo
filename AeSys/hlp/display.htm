Display screen: Pegaesys uses a CRT monitor VGA or Super VGA.

The display includes the following elements:

Line 1: Caption bar, required by Windows, serves as a title and as a report line.

Line 2: MenuBar, required by Windows, accessed using the <Alt> key or mouse.

Line 3: Drawing surface starts here and uses the rest of the screen.

Status line: The line at the top of the drawing area for standing reports, it is transparent to Graphic functions. It contains the following information:

Segment count Trap buffer count Pen color Line style Text Scale @ Zoom Tab length >> angle

Toolbar line: A transparent pseudo bar at the bottom of the screen which serves as a menu for the numeric commands in the current mode. Press the number; dont click the screen. When the mode changes, the toolbar changes to indicates the new commands. Pending operations are indicated by a different color.

Cursor is unique for each GIN mode for instant recognition.

Odometer at screen top right displays the cursor position relative to the working origin.

The display is as uncluttered as possible, maximizing the area available for drawing. The transparent reports can be turned off using F2 F3 and F4, see MenuBar Setup.

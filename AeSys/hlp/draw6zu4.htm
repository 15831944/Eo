Polygon Fill A polygon may be:

Hollowboundary only displayed.

Solidfilled with the current Pen Color.

Patternfilled with any of the suported patterns.

Hatchedfilled with specified crosshatch pattern.

Polygon segments are treated as a single entity.

Fill pattern is specified from the menu bar -Setup-Fill

Regular Polygons can be created using Annotate0-Bubble.

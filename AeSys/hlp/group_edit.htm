Pick & Drag Interactive Edit of a segment.

: (colon) Group Edit Make a segment a dynamic mobile.

This feature provides for dynamic pick and drag of a single group.

Move the cursor to the group and press :. The group will blink once indicating it is mobile. To drag the group, use any cursor positioning method. The group can be dragged using any cursor control and can be edited using a special set of operations. An available commands are displayed on the right side of the status bar.

  • 0 Options (Edit)
  • 2 Rotate Counter-clockwise.
  • 3 Rotate clockwise.
  • 5 Copy and drag the copy.
  • 6 Flip about Mirror axis.
  • 7 Reduce.
  • 8 Enlarge.
  • 9 Modify Attributes to current values.

Press Enter to fix the element and return the primary command mode. Press Esc to discard changes and return to the primary command mode.

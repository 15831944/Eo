Draw Mode Circle

Circles are created using the center and a point on the perimeter.

  • Move cursor to center of circle
  • Press 7 to select the center Circle begins dynamic preview
  • Move the cursor to a point on the perimeter
  • Press 7 to define the radius and create the circle Circle ends dynamic preview
Remarks:
  • Enter aborts creation of a circle.
  • You must use the Draw Mode Ellipse command with equal axes to draw a circle in 3 dimensions.

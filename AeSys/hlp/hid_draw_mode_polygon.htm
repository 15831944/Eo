3 Draw Polygons

Move the cursor to first corner point.

Press 3 to start polygon.

Rubber banding is enabled.

Move cursor to adjacent corner point.

Press 3 to draw first edge.

Move cursor and Press 3 sequentially to outline polygon.

A side is displayed for each entry.

Press ENTER to end a polygon definition.

The polygon will be automatically closed and filled..

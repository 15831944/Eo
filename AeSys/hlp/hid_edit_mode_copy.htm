Edit 5-5 Copies the buffer at the cursor relative to the pivot point.

Press 5 to start the Move.

Rubber banding is enabled.

Move the cursor to the target position.

Rubber band previews the Copy from pivot point to cursor position..

Press 5 to complete the Copy.

The buffer is Copied at the cursor position.

The Copy becomes the Buffer and the Pivot point is reset on the copy so that sequential copies can be made.

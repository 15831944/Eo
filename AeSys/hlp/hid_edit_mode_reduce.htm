Edit 7 Reduces the size of the buffer.

Press 7 to resize the segments in the buffer smaller relative to the pivot point.

The default resize options are 2.0 in the X Y and Z axes.

Pressing the key doubles or halves the size of the segments in the buffer.

The resize ratios for all three axes are reset using Edit 0 - Sizing.

Note Edit 8 undoes Edit 7.

This feature is independent of the cursor position. Dimension entities are not reset. They are simply made larger. Refer to Trap 9- Modify- Dimension for a global reset,

Fixup 3-3 Chamfer to intersect

Move the cursor to line and press 3.

The toolbar number is highlighted.

Move the cursor to other line and press 3

Both lines are cut back from the intersection and connected by a new line.

Use 1 3 instead of 3 3 to trim only the second line

Use 2 3 to trim the first, chamfer the second.

Use 3 2 to chamfer the first, trim the second.

The cutback distance is set using the Fixup 0 key.

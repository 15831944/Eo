Fixup 1 Select a reference line.

The reference line is the target for functions 2, 3, 4, and 6.

Move the cursor to the target line and press 1.

The toolbar number is highlighted.

The selected line is greyed.

Move the cursor to other line and press:

2to trim 2nd line to reference line.

3to chamfer 2nd line to reference.

4to fillet 2nd line to reference line.

6to adjust 2nd line parallel to reference line.

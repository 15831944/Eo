Draw command (Mode menu)

Use this command to select the Draw mode command set. This is the primary command mode used to add primitives to the active layer of the drawing in the active view. You can jump to Draw mode using the keyboard Home or [ keys. All available commands are displayed on the right side of the status bar. Current command is highlighted if neccessary.

Press Numeric Keys to execute:

  • 0 Options
  • 1 Marker
  • 2 2 Lines
  • 3 3 3...[Enter] Polygon
  • 4 4 Quadrilateral
  • 5 5 5 Arc
  • 6 6 6...[Enter] Curve
  • 7 7 Circle
  • 8 8 8 Ellipse
  • 9 9 Insert block

Esc..... cancels the command in progress.

Enter.. initializes or completes a command.

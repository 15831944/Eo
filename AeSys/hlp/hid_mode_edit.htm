Edit Mode - Operate on selected segments

Choose Mode from BarMenu and click Edit, or

Press , or PageDn from any GIN mode.

Press Numeric Keys to execute:

0 Options Edit-Options.

1 Pivot Sets a pivot or reference point for following operations:

2 RotCCW Rotates buffer clockwise.

3 RotCW Rotates buffer counterclockwise.

4-4 Move Moves the buffer to the cursor relative to the pivot point.

5-5 Copy Copies the buffer at the cursor relative to the pivot point.

6 Flip Flips the buffer about the pivot point.

7 Reduce Reduces the size of the buffer.

8 Enlarge Enlarges the size of the buffer.

A buffer is a subset of segments assembled from those in the Workfile.

Trap mode is used to assemble the segments into the buffer.

Edit mode operations allow you to manipulate the subset as a single entity.

The parameters used for the increments of Rotation, Resizing, and the Flip axis are selected from Edit 0 .

There is always a Pivot Point relative to the buffer . It is set automatically when the buffer is trapped. All rotation, translation, sizing, and copying operations are done relative to the pivot point. It is displayed as a marker on the drawing. It can be reset to any point in space.

You can also set the Pivot using the Home - Pivot function.

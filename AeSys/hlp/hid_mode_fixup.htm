Fixup Mode To clean up a drawing

Choose Mode from MenuBar and click Fixup or

Press + from any GIN mode.

Press Numeric Keys to execute:

0 Set options

1 Select reference

2-2 Trim to intersect

3-3 Chamfer

4-4 Fillet arc

5 Square line

1-6 Adjust parallel

The command in progress highlighted on the Toolbar.

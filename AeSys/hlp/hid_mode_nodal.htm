Nodal command (Mode menu)

Use this command to select the Nodal mode command set. This mode is used to alter nodes (individual points) of selected primitives in the active drawing. All available commands are displayed on the right side of the status bar. Current command is highlighted if neccessary.

Select Mode from bar menu and click on Nodal or Press < from any drawing command mode. Press numeric keys to execute a command.

The following keys select the nodes to be operated on:

  • 0AddRemoveSwitch to include or exclude nodes
  • 1PointSelect nodes coincident to cursor
  • 2LinesSelect endpoints of a primitive
  • 3-3AreaSelect nodes inside a rectangular area
  • 8EmptyEmpties the selected nodes list
  • 9EngageSelect node of engaged segment

The following keys operate on the selected nodes:

  • 4-4MoveMove the selected nodes
  • 5-5CopyReplicate groups with selected nodes
  • 6-6ToLineCreate lines from nodes to new cursor position
  • 7-7ToPolygonConverts lines to polygons

Key 0 sets the Node Switch In or Out.

Keys 1, 2, 3, and 9 add nodes to the set when the node switch is set to In.

Keys 1, 2, 3, and 9 remove nodes from the set when the switch is set Out.

After an set of nodes has been selected using 1, 2 or 3, while the Nodal switch is set to "In", you can press 0 and set the Nodal switch to "Out".

You can add and remove nodes interactively until the desired set is composed.

The Nodal buffer is automatically initialized to 0 when you press < . This prevents any residual nodes from being in the buffer. You should only move the cursor using the diamond keys, so that you can un-do a move if you need to. It is advisable to save your work before using this function since there is no mechanism for recovering from an mistake.

Nodal / IN 9 Selects node of engaged segment

Engage a segment on a node by pressing A or Q - Q.

Press 9 to add only the node of the engaged segment.

The node coincident with the cursor is added to the set.

Nodal / OUT 9 UN-Selects node of engaged segment

Engage a segment on a node by pressing A or Q - Q.

Press 9 to REMOVE only the node of the engaged segment.

The node coincident with the cursor is removed from the set.

This operation permits you to be precise in defining the node trap. Only one node at a time is processed. The cursor position and the blinking segment tell you exactly which node you are working with.

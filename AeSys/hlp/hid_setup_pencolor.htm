P Select Pen

Pen Index on the workstation:

Each pen selection is displayed in a unique color and specific linewidth.

256 different pens may be used to create the drawing.

Press P from any Command Mode to change current Pen.

A selection drop-down box is presented. Select the target color from the list.

Default colors:

  • 0 Black (background)
  • 1 Red
  • 2 Green
  • 3 Blue
  • 4 Yellow
  • 5 Cyan
  • 6 Magenta
  • 7 Turquois
  • 8 Gold
  • 9 Dark Red
  • 10 Dark Green
  • 11 Dark Blue
  • 12 Dark Yellow
  • 13 Dark Cyan
  • 14 Dark Magenta
  • 15 Dark Brown

The default colors can be changed by editing the Pen/Color/Definition.txt files in the Program Folder.

Color definition files are formatted as follows:

PenId= RedActive GreenActive BlueActive RedStatic GreenStatic BlueStatic

Active colors are used for Work and Active layers and Opened tracings.

Static colors are used for Static layers and Mapped tracings

The Pen selection determines the weight of the line.

The line width for each pen can be changed by editing the Pen/Width/Definition.txt files in the Program Folder.

Width definition files are formatted in mm. as follows:

0=0.

1=.00752=.0153=.02254=.03

5=.0096=.0167=.02358=.031

9=.007510=.01511=.022512=.03

13=.0614=.1015=.0075

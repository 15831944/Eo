Line Types

Press L from any Drawing Mode.

Line types supported are:

  • 1Solid
  • 2Short dash
  • 3Longdash
  • 4Med dash
  • 5Long Shortdash
  • 6Dash Dot
  • 7Dash Dot Dot
  • 8Dash Dot Dot
  • 9Dotted
  • 10Line shortdash
  • 11Line Dot
  • 12Line Dot Dot
  • 13Line Dot Dot Dot
  • 14Longline Shortdash
  • 15Longline Dot
  • 16Longline Dot Dot
  • 17Longline Dot Dot Dot
  • 18Solid - Solid
  • 19Solid - - Solid
  • 20Solid - - - Solid

The Line Types Dialog is displayed. Up to 20 different line typess may be used to create the drawing.

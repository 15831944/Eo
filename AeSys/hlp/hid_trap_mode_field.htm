Trap 4 Traps all segments with any point in the field.

Move the cursor to one corner of the proposed field.

Press 4 to start a field trap.

Rubber banding is enabled

Move the cursor to the diagonal corner of the field.

Create a rectangle which defines the trap field.

Press 4 again to execute the trap.

All segments within or passing thru the field are added to the buffer.

Note that only rectangular fields can be created.

Entering 2 points on the same axis creates a line and traps all segments which pass through the line.

When Trap 0 - Remove has been selected, this operation removes the target segment from the buffer.

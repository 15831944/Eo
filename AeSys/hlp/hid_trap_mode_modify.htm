Trap 9 Modify

Trap the segments you wish to modify.

Trap must contain segments or an error is returned.

To change attributes of trapped segments to current:

Press 9

Trap Modify Check Boxes

[ ] MarkerText

[ ] Pen[ ] Note

[ ] Line [ ] Height

[ ] Fill[ ] Font

[ ] Dimension

Check desired attributes and Press OK

This command modifies all segments in the trap buffer.

To change current attribute settings, choose MenuBar - Setup.

Trap 2 Traps all segments stitched by a line.

To trap using this option, a line is drawn which passes through the segments you wish to trap.

Move the cursor to the start point of the stitch and press 2 .

Rubber band is enabled.

Move the cursor to the end point of the stitch and press 2 .

All segments which intersect the stitch are placed in the edit buffer.

When Trap 0 - Remove has been selected, this operation removes the target segment from the buffer.

Pegaesys is 3D CADD software for producing construction documents.

This version runs under Microsoft Windows 2000 or later, on IBM compatable PC’s or better with VGA display or better, and can use a Local Area Network. Faster processors, more memory, and bigger screens improve performance. Pegaesys uses the Microsoft Windows family of operating platforms. Windows protocol is used to a point, but IO is unique for speed and accuracy.

PegAESys is complex software, and requires a large number of commands:

Pegaesys divides these commands into Global and Modal functions.

Mode describes the general activity, like Drawing, Dimensioning, Selecting, Annotating ...

Choose any Mode, and the numeric keypad provides 10 commands for that mode.

Global describes a common activity needed for all the modes, like Snaps, Pick, Delete, Move, ....

Global command keys use the typewriter keyboard and are available regardless of the Mode.

Pull down Menus are provided for all commands, but high speed IO is achieved using short-cut keys.

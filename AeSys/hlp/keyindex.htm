Index of Key Commands:

  • A to snap to a point and engage
  • B to jump to the midpoint of engaged segment
  • C to move the cursor DL in the direction DA + 270 deg
  • D to move the cursor Right DL
  • E to move the cursor Up DL
  • F to move the cursor into the screen DL
  • G to move the cursor to a memory home point
  • H to set a memory home point
  • I to jump to the intersection with engaged segment
  • J to jump perpendicular to engaged segment
  • K to jump to segment in specified direction
  • L to select line style
  • M to Mend the endpoint of an engaged segment
  • N to change the attributes of an engaged segment
  • O to access the parameters
  • P to select pen color and weight
  • Q to snap to a segment and engage
  • R to move the cursor DL in the direction DA
  • S to move the cursor Left DL
  • T to jump to surface of polygon
  • U
  • V to move the cursor out of the screen DL
  • W to move the cursor DL in the direction DA + 90 deg
  • X to move the cursor Down DL
  • Y
  • Z to move the cursor DL in the direction DA + 180 deg
  • ? Help
  • = to repaint screen, pan and zoom
  • + to Fixup a drawing
  • ' for lettering
  • " for editing NOTES
  • { used in DDE expressions
  • ] to access the calculator
  • \ to enter a command from GIN mode
  • / to access the files
  • # to move graphic characters into memory WK
  • $ to read graphic characters into memory WKS
  • . to set diamond length
  • > to set diamond angle
  • [ to create segments
  • @ for drawing double lines
  • , to alter segments in TRAP
  • < to alter nodes or end-points of segments
  • | for drawing annotation and dimensioning
  • ~ for drawing dimension lines
  • - to capture segments into an editable buffer
  • _ to sever, divide or cut segments
  • : for dynamic pick and drag
  • ; for dynamic pick and drag of a primitive
  • } used in DDE expressions
  • ` read non-graphic tag data

  • Backspace graphic destructive backspace
  • Insert to restore erased segments in order.

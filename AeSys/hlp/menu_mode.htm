Mode menu commands

The Mode menu offers the following commands:

  • AnnotateSelects the Annotate mode command set
  • CutSelects the Cut mode command set
  • DimensionSelects the Dimension mode command set
  • DrawSelects the Draw mode command set
  • Draw2Selects the Draw2 mode command set
  • EditSelects the Edit mode command set
  • FixupSelects the Fixup mode command set
  • NodalSelects the Nodal mode command set
  • TrapSelects the Trap mode command set

Modes for Graphic Input:

  • Draw [ or Home
  • Annotate | or End
  • Trap - or PageUp
  • Edit , or PageDn
  • 2Line Draw @ or ^Home
  • Dimension ~ or ^End
  • CutUp _ or ^PageUp
  • Nodal < or ^PageDn
  • Fixup +
  • Lettering ‘
  • Revise “
  • Segment Edit :
  • Primitive Edit ;
  • Tasks

GIN (Graphic INput) modes set the primary graphic command functions which are entered using the numeric keypad and/or numbered keys.

To perform graphics operations, the system must be in a GIN MODE.

In each GIN mode, the numeric keys perform a function unique to that mode.

Setup menu commands

The Setup menu offers the following commands:

  • Pen Color P Set Pen Color for drawing
  • Line Type L Set Line Type for the drawing
  • Fill Change Polygon primitive attribute
  • DimLength . Set Skew Diamond Tab Dimension
  • DimAngle > Set Skew Diamond Tab Angle
  • Home Points H = set a point G = go to a point
  • Scale Set scale as ratio Drawing : World
  • Units Choose units of measurement
  • Constraints Change axis and grid constraints
  • Note Change text attributes
  • Global List Global Keys
  • Reset the terminal settings.
  • Color to change Colors on the terminal.
  • Cursor to select a different type of GIN cursor.
  • Mouse to program the mouse buttons.

The operating parameters determine the response of the system to commands and reports. These parameters are installed with default values, but you may change them for a session.

Pick & Drag Interactive Edit of a segment or primitive.

: (colon) SegEdit Make a segment a dynamic mobile.

This feature provides for dynamic pick and drag of a single segment.

; (semicolon) PrimEdit Make a primitive a dynamic mobile.

This feature provides for dynamic pick and drag of a single primitive, even if it is compressed in a segment. Primitive elements remain compressed in segments.

Move the cursor to the segment and press : .

The segment will blink once indicating it is mobile.

To drag the segment, use any GIN cursor method.

The segment can be dragged using any cursor control and can be edited using a special set of operations.

An operation menu is displayed on the toolbar:

0Options (Edit)

2Rotate Counter-clockwise.

3Rotate clockwise.

5Copy and drag the copy.

6Flip about Mirror axis.

7Reduce.

8Enlarge.

9Modify Attributes to current values.

Press Enter to fix the element and return the crosshair.

Press Esc to abort the pick and return to original status.

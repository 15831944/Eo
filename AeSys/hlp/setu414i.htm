Cursor

A unique cursor is provided for each GIN mode. This provides instant recognition of the current mode. This cursor can be manipulated using the arrow keys, a mouse or trackball, or the Diamond keys.

Default The hardware cursors

CrosshairLarge cursor spans the screen,

Symbol To select any segment on the drawing to be used as a cursor, move the crosshair to the desired symbol before changing cursors.

Note that the symbol cursor is a drawing segment and is treated as such in Zoom operations.

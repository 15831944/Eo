Color to change color of a pen for a session:

2 Color Windows are presented in sequence.

Open layers are painted using Color Window 1

First Custom Color is Pen 0, for drawing background.

Define custom colors on Color Window 1 for Pens 0-15.

Add to Custom Colors Press OK to display Window 2.

Mapped layers are painted using Color Window 2.

Define custom colors on Color Window 2 for Pens 0-15.

Add to Custom Colors and Press OK to return to the drawing.

Viewed layers are painted using Pen 0 from Window 1 over Pen 0 from Window 2.

You can assign any color to any pen during a session.

There are 16 default pen colors defined by RGB components in file Pegcols.dat.

You can edit this file to change your default colors at start-up.

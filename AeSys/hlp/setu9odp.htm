View Parameters

Set clip planes using distance from the View Reference Point:

Hitherclip everything this side of the plane

Yonclip everything beyond the plane

Set lighting parameters:

Ambientintensity of ambient light - default = 1

Sourceintensity of point source - default = 1

X distancedistance of source in X dir.

Y distancedistance of source in Y dir.

Z distancedistance of source in Z dir.

Note - distances are in ruler inches:drawing.

Mouse Buttons

In GIN mode, you can program the mouse buttons to send characters. The default settings are:

Left down{ } NULL

Left up{13}

Right down{ } NULL

Right up{27}

The non-ascii keys can be entered using the KeyASCII number enclosed in braces:

backspace {8}

enter {13}

escape {27}

space {32}

For example, you could program:

Left down2to start a line

Left up2{13}to end a line

Right down7 to start a circle

Right up7{13}to end a circle

Left down:to pick and drag

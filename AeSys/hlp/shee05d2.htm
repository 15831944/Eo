Reference management

Files exist as Reference*.LB1 files.

Select a Drawing from the list.:

Choose OK to Get the selected drawing.

Cancel to ABORT

When you Get a reference drawing, it becomes the trap buffer and you can perform Edit Mode operations on it immediately: Move, Copy, Rotate, Flip, Enlarge, or Reduce.

When you Get a reference drawing the Origin of the reference drawing becomes the Pivot of the buffer in the workspace. Thus you can precisely register the imported segments.

Reference drawings are like Tracing Drawings, ie. single layer.

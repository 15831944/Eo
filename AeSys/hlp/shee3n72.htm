Open Open the target file - display drawing.

Choose Open from File Menu.

A dialog box will prompt you through the process. You must “Open” a file to work on it.

You can open any supported file type: .JB0, .JB1, .DXF, .LB1

This command "copies" the target file into the Workfile.

You may add to, delete from, or edit segements in the Open file.

All Trap and Edit functions affect them.

Open files are displayed in bright colors on the screen.

The Name of the Open file is displayed on the caption bar.

Only one file may be open at a time.

Save, or close, the file to make changes permanent on the disk file.

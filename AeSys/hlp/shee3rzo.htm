Current The layer of an open Sheet.JB0 file which receives new segments as they are created..

You may declare any layer of an open sheet Current.

Select target layer, and

Choose Current radio button.

New drawing segements are added to the Current Layer of the Open file as they are created.

You can move segments from Active layers to the Current layer using MenuBar - Cut and Paste

Only one Layer may be current at a time.

One Layer must be current.

By default, the first layer listed is current when the sheet is opened.

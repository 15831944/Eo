Tracing management

Files exist as Tracings(*.JB1).

Select a Tracing from the list.:

Execute enabled file commands on the target.

The following radio buttons are enabled:

OOpenOpen the target for Read-Write

OMapOpen the target for Read

OViewDisplays target in monotone

OCloakErases target from the screen

The following commands are enabled:

Nameto reName a target

Newto Add a new target

Deleteto delete a target

Save Asto continue on a copy

Saveto continue on original

Getto copy target into workfile

Includeto add a Tracing to the Sheet

Cancel to close without saving

OK to save and Close

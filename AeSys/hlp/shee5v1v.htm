Sheets management

Select a Sheet from the drop down list.:

The target Sheet is displayed and highlighted.

The Layers and Tracings for the sheet are displayed.

Execute enabled file commands on the sheet.

Select Layers or Tracings from the lists displayed.

The target Layers and Tracings are highlighted.

Execute enabled file commands on the targets.

Menu for Sheets:

ProjectDisk and File directories.

SheetOperations on selected Sheet.

LayerOperations on selected Layers.

TracingOperations on selected Tracing.

Radio buttons for Layers:

OCurrentreceives new segments

OActiveEditable

OStaticNot editable

OHiddenNot Visible

Command buttons for Layers:

Newto Add a new target

Deleteto delete a target

Changeto change by-layer attributes

Meltto create a Tracing from a Layer

Radio buttons for Tracings:

OOpenOpen the target for Read-Write

OMapOpen the target for Read

OViewDisplays target in monotone

OCloakErases target from the screen

Command buttons for Tracings:

Includeto add a Tracing to the Sheet

Excludeto omit a Tracing from the Sheet

Fuseto create a Layer from a Tracing

Cancelto close without saving

OK to save and Close

When you select a Layer, the segment count, display status, and the by-layer attributes are displayed.

When you select a Tracing, the display status and the Title are displayed.

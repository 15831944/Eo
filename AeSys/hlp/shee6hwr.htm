Cloak Clears a displayed file without saving.

Select targets, and

Choose Cloak radio button.

The file is "unpainted" from the screen.

Open, Mapped, and Viewed files may be Cloaked.

If you choose Cloak for an Open file, the file is discarded, restoring the file as it was when you Opened it.

This lets you abort a contaminated workfile without contaminating the file on the disk. This is equivalent to the Clear Work command from Menu Bar.

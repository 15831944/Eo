Project Menu - Projects are directories of disk files.

NewStart a project - create new directory.

SelectChange projects or directories.

ArchiveFile compression on entire directory.

eXitExit back to Pegaesys.

Projects can be named according to the rules of your operating system. All of the Sheet.JB0 files and Tracing.JB1 files for the project are in the directory for that project. Large projects may be further divided using conventional subdirectories.

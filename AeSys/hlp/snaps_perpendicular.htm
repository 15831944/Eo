J Perpendicular (Jump Snap)

This feature moves the cursor from the current position perpendicularly to a target segment.

With any GIN operation pending :

Move the cursor to the target segment and press J .

The cursor will jump to the perpendicular point on the target segment.

If the engaged segment is a line, and the perpendicular direction is beyond the endpoint of the line, the perpendicular move is to the projection of the engaged line.

For example,

1) start a Line in Draw Mode, enabling rubber banding

2) move the cursor to an existing line and press J

3) the rubber band is now perpendicular to the target line.

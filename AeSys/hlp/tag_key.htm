` (backtick) Tag Key

TAGS are Data receptacles

TAGS consist of a marker and variables with values.

TAGS are created using the ` key or Annotate1:

TAGS are permanent segments in a drawing file.

TAGS can be processed as any other drawing segment:

Move the cursor to the target tag position and

Press ` to evoke the choice menu:

Update presents a dialog box for target tag.

Create inserts a tag at cursor position.

Displayshows the selected tag data on the drawing

Tags carry the following data:

ID Type Color X Y Z TRAP + Var1 Var2 Var3 ... Var64 (your numbered variables).

In a layer or file, the name of a variable is the same for all tags, the values are unique to the tag.

The information in the TAGS can be used for creating reports, application tasks, etc.
